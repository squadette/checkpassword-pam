checkpassword-pam

checkpassword-pam is an implementation of checkpassword-compatible authentication program.

The checkpassword interface is described here: http://cr.yp.to/checkpwd.html

You could use the checkpassword-pam together with:

  • cvs-nserver;
  • qmail
  • mod_auth_external;
  • TMDA;

checkpassword-pam(8) manual page. README. NEWS.
